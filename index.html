📖 프로그램 사용법

■ 전체 진행 흐름

리스팅 목록 → DM 발송 완료/진행 대기 → 계약서발송/수집 → 제품 발송 → 광고 코드/입금정보 수집 → 광고세팅/입금 → 완료

■ 역할 구분

🟡 매니저: 리스팅, DM 발송, 제품 발송, 광고 코드/입금정보 수집

🔴 관리자: 계약서발송/수집, 광고세팅/입금, 정산 완료 처리

■ 인플루언서 추가

1) [인플루언서 관리] 상단에서 제품 탭을 먼저 선택하세요 (없으면 [+ 제품 추가]).

2) [+ 추가] 버튼 → 인스타그램 URL을 붙여넣으면 계정명·릴스 주소가 자동 입력됩니다.

■ DM 발송 단계

· 리스팅 목록에서 DM발송 N 버튼을 누르면 DM 발송 단계로 이동합니다 (다시 누르면 리스팅으로 복귀).

· 인플루언서가 응답하면 상세보기에서 실명·연락처·주소를 입력하세요. 셋 다 입력해야 다음 단계로 넘어갑니다.

· 일반 협찬: [협찬 성사 완료] → 계약서 단계로 진행

· 📦 제품만 협찬: 버튼을 켜면 계약서 단계를 건너뛰고 바로 제품 발송으로 갑니다.

■ 드래그로 이동 (빠른 처리)

· 표의 행을 끌어 위쪽 제품 탭/단계 탭에 놓으면 제품·단계가 바뀝니다.

· 모바일: 행을 길게 누른 뒤 끌어주세요.

· 필수정보(실명·연락처·주소)가 없으면 DM 이후 단계로 이동할 수 없습니다.

■ 제품 발송 단계

상세보기에 실명·연락처·주소가 떠 있으니 [복사] 버튼으로 복사해 택배를 보내고, [제품 발송 완료]를 눌러주세요.

■ 인플루언서 DM 가이드

제품별 대화 가이드 모음입니다. [📋 복사]를 눌러 인스타 DM에 붙여넣어 활용하세요. 가이드 등록·수정은 관리자만 가능합니다.

■ 정산

· DM 발송 1건 = 2,000원 / 협찬 완료 1건 = 20,000원

· 월을 선택해 내역을 확인하고 [클립보드 복사]로 보고할 수 있습니다.

· [정산 완료 처리]는 관리자 로그인 후 가능하며, 완료일·입금액이 기록되어 모든 기기에서 공유됩니다.

■ 기타

· 거절됨: 거절 처리된 인플루언서 보관함 (상세보기에서 복구 가능)

· 🗑 삭제는 복구가 불가능하니 주의하세요.
